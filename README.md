# 💬 MERN Real-Time Chat Application

A full-stack real-time chat application built with the **MERN Stack**. The project focuses on scalable backend architecture, authentication, and instant messaging using modern web technologies.

> **Current Status:** 🚧 Work in Progress

## ✨ Features

✅ User Authentication (JWT)
✅ Secure Password Hashing (bcryptjs)
✅ User Registration & Login
✅ Protected Routes
✅ One-to-One Chat
✅ Conversation-Based Messaging
✅ Message Persistence (MongoDB)

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Krishan-Das/sync-chat.git
```

### Install Dependencies

#### Backend

```bash
cd backend
npm install
```

#### Frontend

```bash
cd frontend
npm install
```

### Environment Variables

Create a `.env` file inside the `backend` folder.

```env
PORT=3000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

### Run the Application

#### Backend

```bash
npm run start
```

#### Frontend

```bash
npm run dev
```
